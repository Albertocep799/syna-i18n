The following is a set of guidelines for contributing to Syna i18n and its related projects.

## Translations

We use **[Crowdin](https://crowdin.com)** to manage all our translations. This ensures consistency and makes it easier for everyone to contribute.

> [!IMPORTANT]
> **Please do not submit Pull Requests with translation changes directly to this repository.**
> All translations are synced from Crowdin. Changes made manually here may be overwritten.

### How to Contribute

1.  **Create a Crowdin Account:** If you don't have one, sign up at [Crowdin](https://accounts.crowdin.com/login).
2.  **Join the Project:** Access our project on Crowdin: **[LINK TO CROWDIN PROJECT]**
3.  **Start Translating:** Select your language and start suggesting translations.

### Proofreading

If you are a native speaker or experienced translator and would like to become a **Proofreader** (to approve translations), please let us know on our **[Discord Server](https://discord.gg/E6JrWfeqkQ)**.

## Issues and Bugs

If you find a typo in the English source strings or a technical issue with the JSON files:

*   Open an [Issue](https://github.com/Albertocep799/syna-i18n/issues) on this repository.
*   Or report it in the `#translations` channel on our [Discord Server](https://discord.gg/E6JrWfeqkQ).

## Code of Conduct

This project and everyone participating in it is governed by the [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.
